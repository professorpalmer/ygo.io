YGO Purrely Combos








Relevant Combos

Noir + Leap (Sleepy Memory)

1. Use Sleepy, discard 1, SS Black Cat.
2. Cat effect to add MFP.
3. MFP activate and search 2x Sleepy, 1x Delicious.
4. Activate Black Cat to target sleepy from GY. Turn to Epur Noir.
5. Use Sleepy Memory in hand, chain Noir to absorb. Set leap.

Noir + ~ (My Friend Purrely, 1 Memory)

1. MFP, search 3x Street.
2. Activate Street, activate QP spell in hand.
3. SS Black Cat, search White Cat.
4. NS White Cat, search for luck.
5. Target QP spell in GY on Black Cat for board break.

Noir + ~ (My Friend Purrely)

1. Search 3x Sleepy.
2. Sleepy to SS Black Cat.
3. If you are interrupted, any hand trap body NS will make Little Knight S:P.
4. Black Cat search White Cat.
5. NS White Cat, search for luck.

Noir + S:P (1 Normal Summon, 1 Memory)

1. Black Cat searches MFP, White Cat searches any QP spell you can get.
2. If Delicious not in hand, use MFP to search it instead of Sleepy x3.
3. NS Cat. Activate effect for extra card.
4. Use Memory from hand, discard 1.
5. SS other cat, activate effect for extra card.
6. Set MFP. Black Cat effect > Delicious in GY > Plump.
7. Swap #4 and #5 if no Delicious open.
8. Use MFP, search Sleepy x3 if didn't start with Delicious.
9. Activate Memory, chain Plump. Discard once again and SS White Cat for search.
10. At this point, chain as many Memory as possible. Ignore discards.
11. Use Plump effect to attach 2 spells/traps.
12. Plump > Expur Noir.
13. White Cat + White Cat > S:P Little Knight / Robin.

Noir + Any other Purrely XYZ (Purrely, Purrelyly, 2 Memories)

1. NS White Cat. Grab free spell.
2. Use a Memory. Discard 1 to SS Black Cat. Search MFP.
3. MFP 3x/2x Sleepy, 1x Delicious.
4. If you haven't used a Sleepy already, use one now and do not discard, just to get in the GY. Alternatively, you can use White Cat for Epur Noir and Black Cat for Plump/Beauty.
5. Black Cat effect > target Sleepy in GY for Epur Noir.
6. Use Memory in hand, absorb with Epur Noir, set Leap.
7. Use any Memory in hand with the White Cat effect to get another Epur body. (Plump/Beauty/Noir).
8. End turn.

Noir (4) + Leap (Sleepy Memory, Any Memory)

1. Use Sleepy, summon Black Cat.
2. Search MFP.
3. Activate MFP, search 3x Street.
4. Activate Street.
5. Activate Black Cat, target Sleepy in GY, Epur Noir.
6. Use extra Memory, chain Noir to set Leap.
7. Field Spell Sleepy Memory for more draw.
